# 💰 Business Funding Platform

A modern, responsive fintech website built completely using **HTML5, CSS3, and Vanilla JavaScript**.

This project demonstrates how a production-inspired financial services platform can be developed without using any frameworks or external libraries. The website focuses on helping businesses understand, compare, and estimate different funding solutions through an intuitive and user-friendly interface.

Developed by **Prince Thakur** as a frontend development project.

---

# 📌 Project Overview

The Business Funding Platform is a complete multi-page website inspired by modern fintech products.

It allows businesses to:

- Explore different funding solutions
- Learn about financing products
- Calculate funding eligibility
- Estimate EMI and repayment
- Browse educational finance resources
- Understand funding workflow
- Connect through a responsive contact page

The entire application is built using only:

- HTML
- CSS
- JavaScript

No frameworks, no libraries, no backend.
